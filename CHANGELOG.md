# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-03-26

### Added
- Phase 1: Authentication (OAuth 2.0), customers, jobs, invoices, technicians
- Phase 2: Memberships, estimates, leads, pricebook, dispatch, reporting, revenue intelligence, snapshot, API escape hatch
- Phase 3: Write operations — create/update customers, book/update/cancel/complete jobs, assign dispatch, manage leads and bookings
- Shell completion installer
- Confirmation prompts and --dry-run mode for all mutating commands
- Agent mode (ST_AGENT_MODE=1) for compact JSON output
- Table, JSON, and CSV output formats
- --fields flag for selective field output on all list commands
